Conversion Math

API rate format: unitsPerUSD  (e.g. 0.000213 oz/USD for gold)

PRECIOUS  (XAU XAG XPT XPD):  usdPerOunce = 1/rate  [troy oz: ÷31.1034768]
INDUSTRIAL (XCU NI):           usdPerOunce = 1/rate  [avdp oz: ÷28.349523]
                                → usdPerPound = usdPerGram × 453.59237
XCO:                           same industrial chain
                                → usdPerMetricTon = usdPerGram × 1_000_000
OIL (BRENT WTI):               usdPerBarrel = 1/rate  (direct, no gram step)
FX  (BRL EUR CAD):             fxUsdXxx = rate  (direct exchange rate)

Display Adjustments

XCU and NI: fmt(v) = $(v - 0.15).toFixed(2) — 15¢/lb visual reduction applied before display.
Also propagated into subLabel: Math.round((v - 0.15) × 2204.623)

Startup Seed

seedHistory() IIFE:
  SEED_DEPTH_DAYS = 1825 (5 years)
  CHUNK_DAYS = 365 (API max window)

  1. hydrateTimeseries(disk → memory)
  2. Compute oldest/newest in-memory
  3. Build date ranges to fill gaps:
     – backfill (older missing data)
     – top-up (newer missing data)
  4. For each range:
     fetchTimeframe(start, end, all syms)
     → ingestChunk(rates)
     → append to timeseriesBySymbol
     500ms delay between chunks
  5. seedComplete = true
ingestChunk applies the same conversion math per symbol type (OIL / FX / metal). New symbol types must be added to BOTH refreshAllSymbols AND ingestChunk.

News Pipeline

Sources

Source Requires Key
newsdata.io YES — NEWS_API_KEY
Kitco RSS no
Mining.com RSS no
CNBC Commodities RSS no

Processing

fetchNews(apiKey):
  1. Parallel: 2 newsdata.io queries
     (if apiKey set)
  2. Parallel: 3 RSS fetches (always)
  3. Map → NewsItem
     · decodeEntities(title)
     · decodeEntities(description)
     · parsePubDate (auto-detects format)
     · tagArticle (METAL_KEYWORDS match)
  4. Dedupe by normalized title
  5. Filter by RELEVANCE_TERMS
  6. Sort by pubMs desc
  7. Slice to 30 items

NewsItem Shape

id:          base64(link).slice(0,16)
title:       string (entities decoded)
description: string (200 char max)
link:        string
source:      string (source_id)
pubDate:     string (original)
pubMs:       number (UTC epoch ms)
tags:        string[] (symbol keys)

pubDate Parsing

newsdata.io: "YYYY-MM-DD HH:MM:SS"
→ replace ' ' with 'T', append 'Z'
→ force UTC parse

RSS: RFC 2822 (auto, no transform)

Known Gotchas

border + borderLeft hover bug
Never use border shorthand alongside borderLeft when there's a transition: border-color. Use borderTop / borderRight / borderBottom + borderLeft separately.
pubDate timezone
newsdata.io returns "YYYY-MM-DD HH:MM:SS" with NO timezone. Must append "T" + "Z" before Date.parse() or articles show 3h stale on UTC-3 machines.
Sparkline double-fire
Call paint() synchronously first. ResizeObserver should only re-trigger when canvas.width !== Math.round(newW * dpr). Otherwise the two calls cancel each other's rAF mid-animation → blank canvas.
noUncheckedIndexedAccess
RegExpMatchArray[n] is string|undefined. Always ?? "" on regex capture groups.
Static handler order
All app.get/post('/api/...') must be registered BEFORE app.use(express.static(...)).

Production Hardening (2026-05-26)

Resilience Patterns

Pattern Why
--min-instances 1 Keeps container warm — no cold-start blank-chart window
Promise.race 30s guard AbortSignal.timeout in undici can miss the response stream; this is the backstop
Self-rescheduling refresh Failure → 60s retry; success → 5min cycle. No setInterval drift.
Per-chunk seed catch One bad chunk no longer kills the whole seed
503 + Retry-After Empty cache AND empty timeseries → 503. Otherwise serve historical.
Structured JSON logs Cloud Logging picks up severity field automatically

Tunable Timeouts

fetchLatest:    20s  (AbortSignal)
fetchChange:    20s  (AbortSignal)
fetchTimeframe: 60s  (AbortSignal — heavy payload)
coalesceRefresh: 30s  (Promise.race manual timer)
READY_TIMEOUT:  10s  (boot grace window)
RETRY_INTERVAL: 60s  (after failed refresh)
REFRESH_INTERVAL: 5min (METALPRICE_REFRESH_MINUTES=5)
