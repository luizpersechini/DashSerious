# Dashboard 2 — Financial data dashboard (gold, cobalt prices)

## STOP — read this before pushing to main

Pushes to `main` deploy to production Cloud Run immediately. Before any change to `src/`, `public/`, or `.github/workflows/`:

1. Read `QA.md` (root)
2. Walk through `qa/CHECKLIST.md` for the area you touched
3. Run the relevant tests in `qa/SMOKE-TESTS.md` locally AND post-deploy
4. Scan `qa/KNOWN-BUGS.md` for prior bugs in the area you're touching

**Producer ≠ consumer rule:** "the server returns the right thing" does NOT verify "the user sees the right thing." Always close the loop with a browser-side or end-to-end check.

## Stack

TypeScript · Node.js · Express · vitest · lightweight-charts · xlsx

## Run

```
npm run dev           # dev server (tsx, hot reload)
npm run fetchGold     # fetch latest gold prices
npm run export:cobalt # export cobalt data to xlsx
```

## Build & start

```
npm run build   # tsc → dist/
npm start       # node dist/server.js
```

## Test

```
npm test             # vitest run (single pass)
npm run test:watch   # vitest watch mode
```

## Structure

- `src/server.ts` — Express entry point
- `src/metalpriceClient.ts` — price API client
- `src/storage.ts` — data persistence
- `src/config.ts` — env/config
- `scripts/` — standalone data-fetch scripts
- `dist/` — compiled output, don't edit

## Don't touch
